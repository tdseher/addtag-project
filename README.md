# CRISPR AddTag README #

Program for identifying unique endogenous gRNA sites and creating unique synthetic gRNA sites.

### Requirements ###

* [Python 3.5](https://www.python.org/downloads/)
* [regex module](https://pypi.python.org/pypi/regex)
